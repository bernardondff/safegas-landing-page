Como funciona

Hardware → Nuvem → Você avisado.

Três camadas integradas que trabalham juntas 24/7, sem depender da sua presença.

🔧

Hardware IoT

Sensor MQ-6 de alta sensibilidade detecta concentrações a partir de 100 ppm. Microcontrolador ESP32 processa e envia dados continuamente via Wi-Fi.

ESP32 MQ-6 DHT22 Bateria backup

Nuvem em tempo real

Dados chegam ao Firebase em milissegundos. Regras de alerta são processadas na nuvem — disponíveis em qualquer dispositivo.

Firebase Realtime DB Cloud Functions HTTPS

Alerta no celular

Push notification instantâneo quando qualquer leitura ultrapassa o limiar seguro. Dashboard com histórico, gráficos e relatórios.
